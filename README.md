GATE Preparation Dashboard

A personal web-based dashboard designed to organize and track GATE (ECE) exam preparation.

This project focuses on clarity, responsiveness, and practicality rather than flashy frameworks.

Built using HTML, CSS, and vanilla JavaScript.
